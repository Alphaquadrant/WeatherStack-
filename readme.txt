 * This is an weather application based on Node js. This accesses live data from mapbox.com and darksky.net using 
 their specially made APIs. Then it displays the data.

 This project is made by Vaishav Pande (202010019) and Yash Kulkarni (202010020)


::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::
|||||||||||||||||||||||||||||||||||||||||||     System Requirements     ||||||||||||||||||||||||||||||||||||||||||||||
::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::

  -  1 GB ram minimumm
  -  Node package  manager installed
  -  Any internet browser


::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::
||||||||||||||||||||||||||||||||     Installing the project   |||||||||||||||||||||||||||||||||||||
::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::

  - Extract the contents of the compressed file to any folder.
  - Have node package manager installed.

You do not need to do any additional steps to run this program.


::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::
||||||||||||||||||||||||||||||||||||||||     Running the app    |||||||||||||||||||||||||||||||||||||||||||
::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::

  - Open the extracted folder. Navigate to WebServer\src\
  - Open command prompt and change directory to this src folder.
  - Type "Node app.js". Console will display following message
            "Server is up on port 3000"
  - Open any internet browser you have, and paste the following URL in address bar
            "http://localhost:3000"
